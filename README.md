# 🏀 Basketball Scoreboard

A simple and interactive **Basketball Scoreboard Web App** built using **HTML, CSS, and JavaScript**.
This project allows users to track scores for **Home** and **Away** teams with buttons to add points or reset the score.

---

## 🚀 Live Demo

Try the app here:

🔗 https://basketballscorecard007.netlify.app/

---

## 📌 Features

* Add **+1, +2, or +3 points** to the Home team
* Add **+1, +2, or +3 points** to the Away team
* **Reset score** for each team
* Simple and responsive layout
